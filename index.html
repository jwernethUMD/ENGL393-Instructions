Creating a Website Instructions

Writing the HTML of your website

  1. Select new file
  2. Rename the file to index.html
  3. Change the title tag
  4. Type Instructions Section in between the body tags
  5. Put h1 tags around the text
  6. Use h2 tags for any subcategory names
    1. Use ol tags inside li for sublists
    2. Change attribute to use letters
  7. img tags for images
  8. Assemble your website!
